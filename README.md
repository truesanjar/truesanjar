## 🚀 **Насб ва иҷро**

```bash
# Клон кардани репозитори
git clone https://github.com/yourusername/instagram-clone.git

# Гузариш ба папкаи лоиҳа
cd instagram-clone

# Насб кардани вобастагиҳо
npm install

# Иҷро кардан дар муҳити таҳия
npm run dev

# Сохтани версияи production
npm run build
